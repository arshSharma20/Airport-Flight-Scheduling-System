# Airport Flight Scheduling System

A Java-based Airport Flight Scheduling System that manages and prioritizes
flights using a min-heap data structure.

## Features
- Flight scheduling based on departure time and priority
- Min-heap implementation from scratch
- Comparable-based ordering
- Heap sort using merge sort for end-of-day scheduling
- Interactive console menu

## Technologies
- Java
- Data Structures (Min Heap, Arrays)
- Comparable Interface
- Merge Sort
- Object-Oriented Programming
